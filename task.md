## ✅ ما تم إنجازه (DONE)

### 🆕 تحديث ٢٠٢٦-٠٦-١٣ — اختيار المنتجات + PDF مخصص + تحسين الجوال
- **اختيار المنتجات (Checkbox):** الضغط على صورة أي منتج يحدّده (✓ بزاوية البطاقة + إطار مميّز). أزرار «تحديد القسم» في رأس كل قسم و«تحديد الظاهر» في الشريط. التحديد يُحفظ في `localStorage` (`elaf_sel`).
- **شريط تحديد سفلي ثابت** يظهر عند اختيار منتج واحد فأكثر: عدد المختار + «مسح التحديد» + «عرض المختار» (فلتر) + «إنشاء PDF للمختار».
- **توليد PDF داخل المتصفح من التحديد:** محرّك ترقيم صفحات A4 بلغة JS (نُقل من `generate_print.py`) يبني غلافًا فاخرًا + فهرسًا + بانرات أقسام + بطاقات (صورة/اسم/SKU/سعر) + ترويسة/تذييل بأرقام صفحات + غلاف ختامي، ثم `window.print()` → «حفظ كملف PDF». يعمل على الكمبيوتر والجوال، ويستخدم الأسعار المعدّلة محليًا. إن لم يُختر شيء يُولَّد الكتالوج الكامل.
- **تحسين الجوال:** ترويسة مدمجة (إخفاء النص الإنجليزي + زر PDF أيقونة)، شريط أقسام أفقي قابل للتمرير، شبكة بعمودين، أهداف لمس أكبر للخانات، شريط سفلي يراعي `safe-area-inset` في iOS.
- **تحسينات بصرية:** لمسة ذهبية (#c8a25c) في الغلاف والأزرار، تأثيرات تحديد، شاشة «جارٍ تجهيز PDF» مع تحميل الصور قبل الطباعة، تنبيهات (toast).
- التحقق: لُقطات شاشة (كمبيوتر/جوال) + توليد PDF تجريبي (١٢٠ صفحة، صور مضمّنة) + اختبار منطق الترقيم للتحديدات الصغيرة.

- **٩٦١ منتج** كلها باسم شركة ايلاف الشرقية التجارية، مُصنّفة إلى **١٤ قسمًا** (المائدة والتقديم، الطهي، الأكواب، القوارير، الحفظ والتنظيم، الديكور، التنظيف، الورقيات، الأحذية، الرياضة، الألعاب، الحيوانات، العدد والأدوات، متنوعة).
- **مصادر البيانات:**
  - كتالوج الطباعة الرسمي (`كتالوج إيلاف طباعة 2026-1.pdf`): استخراج آلي للاسم + رقم الصنف + **السعر** + **الصورة عالية الجودة** (نحو ٢٤١ منتج بصور وأسعار حقيقية).
  - قوائم المنتجات الإضافية (إكسل): الاسم + رقم الصنف + الصورة، وتم تطبيع الأسماء العربية وتصنيفها.
- **رمز الريال السعودي الرسمي** (SVG) بجانب كل سعر.
- **كتالوج PDF** (`كتالوج ايلاف الشرقية 2026-2027.pdf`، ١١٩ صفحة A4): غلاف فاخر + فهرس قابل للنقر بأرقام الصفحات + فواصل أقسام + ترويسة/تذييل + بطاقات (صورة + اسم + SKU + السعر).
- **كتالوج ويب تفاعلي** (`index.html`): أقسام + تنقل بين الأقسام + بحث فوري + صور + أسعار + خانات أسعار قابلة للتعديل (تُحفظ محليًا) + متجاوب.
- الشعار (logo-full / trans / white)، خطوط Cairo + Tajawal، RTL.
- **منشور على GitHub + GitHub Pages** (مستودع عام).

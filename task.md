# مهمة كتالوج إيلاف الشرقية — Elaf Al Sharqiah Catalog

آخر تحديث: 2026-06-06

كتالوج منتجات لموسم ٢٠٢٦ / ٢٠٢٧ — **شركتان: ايلاف الشرقية (٢١٩) + قوين (٧٤٢) = ٩٦١ منتج**. نسخة PDF احترافية ونسخة ويب تفاعلية.

## 🌐 مباشر: https://gzjz10.github.io/elaf_cataloge/

## 🆕 آخر إضافة: منتجات شركة قوين (٧٤٢)
- استُخرجت من `منتجات شركه قوين 2026-2027.xlsx` (الاسم + رقم الصنف + الوحدة + الصورة).
- الصور: **٢٢ منتج** حصلت على صورة عالية الجودة من كتالوج الطباعة (مطابقة بالـSKU/الاسم)؛ **٧٠١** تستخدم الصورة المصغّرة من الإكسل (مع شحذ)؛ **١٩** بدون صورة.
- **لا توجد أسعار لمنتجات قوين** في الإكسل (خانات ر.س فارغة، قابلة للتعبئة في الويب).
- الكتالوج الآن مقسوم لجزأين (ايلاف تركوازي / قوين بنفسجي) مع فلترة حسب الشركة في الويب وفاصل لكل شركة في الـPDF.
- **هنا تعود فكرة Higgsfield**: لرفع جودة صور قوين الـ٧٠١ (تحتاج عودة خدمة Higgsfield + رصيد كافٍ).

## ✅ ما تم إنجازه (DONE)

- **مصدر البيانات النهائي = كتالوج الطباعة الرسمي** (`كتالوج إيلاف طباعة 2026-1.pdf`):
  - استخراج **٢١٩ منتج** آليًا من ملف PDF (٨١ صفحة) عبر PyMuPDF.
  - لكل منتج: **الاسم + رقم الصنف (SKU) + السعر + الكمية بالكرتون + الصورة الحقيقية عالية الجودة**.
  - الأسعار الحقيقية بالريال السعودي (من ١.٥ إلى ٥٤٠ ر.س).
  - تنظيف الأسماء العربية (تحويل أحرف العرض Presentation Forms إلى نص قياسي + ترتيب RTL).
- **تصنيف المنتجات** إلى **١٠ أقسام**: المائدة والتقديم (٨٨)، الفازات والديكور (٣١)، التنظيف (٢٧)، الورقيات والمناديل (٢٠)، القوارير والمطارات (١٨)، الحفظ والتخزين (١٥)، الطهي (٨)، الرياضة والكرات (٦)، الأكواب (٥)، متنوعة (١).
- **رمز الريال السعودي الرسمي** (SVG) بجانب كل سعر.
- **كتالوج PDF** (`كتالوج ايلاف الشرقية 2026-2027.pdf`، ٣٤ صفحة A4):
  - غلاف فاخر (شعار أبيض على تركوازي متدرج) + فهرس **قابل للنقر** بأرقام الصفحات.
  - فواصل أقسام، ترويسة (شعار + اسم القسم)، تذييل بأرقام الصفحات.
  - بطاقات: صورة حقيقية + اسم + شارة SKU + **السعر بالريال**.
  - خطوط Cairo + Tajawal، RTL، جاهز للطباعة، نص قابل للتحديد.
- **كتالوج ويب تفاعلي** (`index.html`): أقسام + شريط تنقل بين الأقسام + بحث فوري + الصور والأسعار الحقيقية + خانات سعر **قابلة للتعديل** (تُحفظ محليًا) + عرض الكمية بالكرتون + متجاوب للجوال.
- **الشعار**: من `لوقو ايلاف.pdf` → `logo-full / logo-trans / logo-white`.
- إزالة الشريط الأبجدي (استُبدل بأقسام)، وتنظيف الملفات المؤقتة/القديمة.

### ملفات المشروع
```
catalog/
  كتالوج ايلاف الشرقية 2026-2027.pdf   ← التسليم الرئيسي (PDF)
  index.html                            ← الكتالوج التفاعلي (ويب)
  print.html                            ← مصدر الـ PDF (يُولّد آليًا)
  extract_print.py                      ← استخراج الصور+الأسعار من PDF الطباعة
  generate_print.py                     ← مولّد ملف PDF
  data/products.json + products.js      ← بيانات المنتجات + التصنيفات
  assets/brand/  +  assets/print/        ← الشعارات + صور المنتجات
task.md  ← هذا الملف
```

- **تصدير PDF منفصل لكل قسم** (للمندوبين).
- **صفحة معلومات الشركة والتواصل** على الغلاف الخلفي + خريطة/أرقام.
- **فهرس PDF داخلي (bookmarks)** حقيقي بالإضافة للروابط.

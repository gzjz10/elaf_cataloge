## ✅ ما تم إنجازه (DONE)

- **٩٦١ منتج** كلها باسم شركة ايلاف الشرقية التجارية، مُصنّفة إلى **١٤ قسمًا** (المائدة والتقديم، الطهي، الأكواب، القوارير، الحفظ والتنظيم، الديكور، التنظيف، الورقيات، الأحذية، الرياضة، الألعاب، الحيوانات، العدد والأدوات، متنوعة).
- **مصادر البيانات:**
  - كتالوج الطباعة الرسمي (`كتالوج إيلاف طباعة 2026-1.pdf`): استخراج آلي للاسم + رقم الصنف + **السعر** + **الصورة عالية الجودة** (نحو ٢٤١ منتج بصور وأسعار حقيقية).
  - قوائم المنتجات الإضافية (إكسل): الاسم + رقم الصنف + الصورة، وتم تطبيع الأسماء العربية وتصنيفها.
- **رمز الريال السعودي الرسمي** (SVG) بجانب كل سعر.
- **كتالوج PDF** (`كتالوج ايلاف الشرقية 2026-2027.pdf`، ١١٩ صفحة A4): غلاف فاخر + فهرس قابل للنقر بأرقام الصفحات + فواصل أقسام + ترويسة/تذييل + بطاقات (صورة + اسم + SKU + السعر).
- **كتالوج ويب تفاعلي** (`index.html`): أقسام + تنقل بين الأقسام + بحث فوري + صور + أسعار + خانات أسعار قابلة للتعديل (تُحفظ محليًا) + متجاوب.
- الشعار (logo-full / trans / white)، خطوط Cairo + Tajawal، RTL.
- **منشور على GitHub + GitHub Pages** (مستودع عام).

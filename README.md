# كتالوج شركة ايلاف الشرقية التجارية | Elaf Al Sharqiah Trading — Product Catalog

كتالوج منتجات موسم **٢٠٢٦ / ٢٠٢٧** — نسخة ويب تفاعلية + نسخة PDF جاهزة للطباعة.
**٩٦١ منتج** من شركتين: **ايلاف الشرقية (٢١٩)** بالصور والأسعار الحقيقية + **قوين (٧٤٢)**، مع فلترة حسب الشركة.

## 🌐 المعاينة المباشرة (Live)
- **الكتالوج التفاعلي:** https://gzjz10.github.io/elaf_cataloge/
- **ملف PDF:** [`catalog/كتالوج ايلاف الشرقية 2026-2027.pdf`](catalog/)

## ✨ المميزات
- اتجاه عربي RTL، خطوط Cairo + Tajawal.
- ١٠ أقسام منتجات + بحث فوري + تنقل بين الأقسام.
- صور حقيقية عالية الجودة + الأسعار بالريال السعودي (رمز ﷼ الرسمي).
- خانات أسعار قابلة للتعديل في الويب (تُحفظ محليًا).
- نسخة PDF احترافية (غلاف، فهرس قابل للنقر، أرقام صفحات).

## 📁 البنية
```
index.html                      ← تحويل إلى الكتالوج
catalog/
  index.html                    ← الكتالوج التفاعلي
  كتالوج ايلاف الشرقية 2026-2027.pdf
  print.html                    ← مصدر الـ PDF
  data/products.json|js         ← البيانات + التصنيفات
  assets/brand | assets/print   ← الشعارات + صور المنتجات
  extract_print.py              ← استخراج الصور+الأسعار من PDF الطباعة
  generate_print.py             ← توليد ملف PDF
task.md                         ← سجل المهام (تم / المتبقي / أفكار)
```

## 🔧 إعادة التوليد
```bash
python3 catalog/extract_print.py     # استخراج الصور والأسعار من PDF الطباعة
python3 catalog/generate_print.py    # توليد print.html
# ثم: طباعة print.html إلى PDF عبر Chrome --print-to-pdf
```

> ملاحظة: ملفات المصدر الكبيرة (PDF الطباعة 125MB، وملفات الإكسل) غير مرفوعة في المستودع لتجاوزها حدود الحجم.

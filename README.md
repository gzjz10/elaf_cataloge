# كتالوج شركة ايلاف الشرقية التجارية | Elaf Al Sharqiah Trading — Product Catalog

كتالوج منتجات موسم **٢٠٢٦ / ٢٠٢٧** — **٩٦١ منتج** ضمن **١٤ قسمًا**. نسخة ويب تفاعلية + نسخة PDF جاهزة للطباعة.

## 🌐 المعاينة المباشرة (Live)
- **الكتالوج التفاعلي:** https://gzjz10.github.io/elaf_cataloge/
- **ملف PDF:** [`catalog/كتالوج ايلاف الشرقية 2026-2027.pdf`](catalog/)

## ✨ المميزات
- اتجاه عربي RTL، خطوط Cairo + Tajawal.
- ١٤ قسم منتجات + بحث فوري + تنقل بين الأقسام.
- صور المنتجات + الأسعار بالريال السعودي (رمز ﷼ الرسمي).
- خانات أسعار قابلة للتعديل في الويب (تُحفظ محليًا).
- **اختيار المنتجات** بالضغط على صورة أي منتج (✓)، أو «تحديد القسم» / «تحديد الظاهر»، مع شريط تحديد ثابت وفلتر «المختار فقط». التحديد يُحفظ محليًا.
- **إنشاء PDF مخصص من داخل المتصفح** يحتوي المنتجات المختارة فقط (غلاف فاخر + فهرس + بانرات أقسام + أرقام صفحات + غلاف ختامي) — يعمل على الجوال أيضًا عبر «حفظ كملف PDF».
- **تصميم متجاوب للجوال**: ترويسة مدمجة، شريط أقسام أفقي، شبكة بطاقات بعمودين، شريط تحديد سفلي يراعي المساحة الآمنة في iOS.
- نسخة PDF كاملة جاهزة للطباعة (غلاف، فهرس قابل للنقر، أرقام صفحات) — تُولّد آليًا أيضًا من `generate_print.py`.

## 📁 البنية
```
index.html                      ← تحويل إلى الكتالوج
catalog/
  index.html                    ← الكتالوج التفاعلي
  كتالوج ايلاف الشرقية 2026-2027.pdf
  print.html                    ← مصدر الـ PDF
  data/products.json|js         ← البيانات + التصنيفات
  assets/brand | assets/print | assets/items
  extract_print.py              ← استخراج الصور+الأسعار من كتالوج الطباعة
  generate_print.py             ← توليد ملف PDF
task.md                         ← سجل المهام (تم / المتبقي / أفكار)
```

## 🔧 إعادة التوليد
```bash
python3 catalog/generate_print.py
# ثم طباعة print.html إلى PDF عبر Chrome --print-to-pdf
```

> ملاحظة: ملفات المصدر الكبيرة (كتالوج الطباعة 125MB وملفات الإكسل) غير مرفوعة لتجاوزها حدود الحجم.
